
=head1 NAME

Nes::Obj::date - Get date and time.

=head1 SYNOPSIS

    {: include ( '{: * cfg_obj_top_dir :}/date.nhtml', 
                 'gmt',                  {: # local o gmt :}
                 '%A %e %B %Y %H:%M:%S'  {: # format :}
               ) 
    :}

=head1 DESCRIPTION

Get date and time in POSIX strftime format.

=head1 PARAMETERS

=over 2

=item local o gmt

Get local or gmt.

=item format

POSIX strftime format 

=back
